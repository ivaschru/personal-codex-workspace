## Claude Code

- Канонические проектные навыки находятся в `skills/<name>/SKILL.md`. Когда запрос соответствует описанному в `AGENTS.md` навыку, полностью прочитай соответствующий `SKILL.md` перед действиями.
- Встроенные инструменты разных агентских сред могут называться по-разному. Следуй цели и границам процедуры, используя доступный безопасный браузер, MCP, CLI или файловые инструменты.
- Не дублируй правила из `AGENTS.md` в этом файле: импорт выше сохраняет единый источник инструкций для Codex, Claude Code и других агентов.
